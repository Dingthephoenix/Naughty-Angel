# 贪吃蛇（Web）

一个基础版贪吃蛇小游戏，使用原生 HTML/CSS/JavaScript 与 `canvas`。

## 运行

- 直接用浏览器打开 `index.html` 即可游玩（无需构建/打包）
- 如果遇到个别浏览器对本地脚本的限制，也可以用任意静态服务器打开（可选）

## 操作

- **方向键 / WASD**：控制方向
- **空格**：暂停/继续
- **R**：重开
- **开始/重开按钮**：开始或重置

## 规则

- 吃到食物：蛇长度 +1，分数 +1
- 撞墙或撞到自己：游戏结束

## 结构

- `index.html`：页面与 UI
- `styles.css`：样式
- `src/main.js`：入口（循环与事件）
- `src/game.js`：状态与规则（tick）
- `src/render.js`：绘制
- `src/input.js`：输入处理
